# Bag Files Directory

## 📁 Purpose

This directory contains ROS bag files for offline processing with FAST-LIVO2.

## 🎯 Usage

Place your `.bag` files here to use with the bag processing system:

```bash
# Structure should be:
docker/bag-processing/bags/
├── README.md (this file)
├── your_dataset_1.bag
├── your_dataset_2.bag
└── ...
